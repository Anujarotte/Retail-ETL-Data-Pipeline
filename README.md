# Retail Data Warehouse & ETL Pipeline

## 📌 Overview
Built an end-to-end ETL pipeline to process retail sales data and load it into a MySQL data warehouse.

## ⚙️ Tech Stack
- Python (Pandas)
- MySQL
- SQLAlchemy
- PyMySQL

## 🔄 Pipeline Steps
1. Extract data from CSV
2. Transform data (cleaning, feature engineering)
3. Load data into MySQL

## 🏗️ Data Model
- Fact Table: fact_sales
- Dimension Tables: dim_products, dim_customers

## 📊 Dataset
- 160,000+ retail transactions

## 🚀 Outcome
Efficient data pipeline enabling structured storage and analysis of retail data.
